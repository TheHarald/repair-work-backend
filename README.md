## Backend приложение для системы ремонтных заявок.

### Установка

```
npm install
```

##### .env

Создать и заполнить .env

```
JWT_SECRET=
DB_USERNAME=
DB_PASSWORD=
DATABASE=
```

### Запуск

```
npm run dev
Открыть http://localhost:3001/api/
```

### Технологии

- Express
- Sequelize
- MySQL
- JWT
